Velkommen

Dette er en enkel nettside jeg utvikler for tiden for å vise noe av det jeg kan.

  • Du kan trykke på meny-ikonet oppe til venstre og få tre valg.
  • Du lage brukere og se en oversikt over dem. Disse blir lagret på backend og hentet med api-kall.
  • Du kan og trykke på login oppe til høyre. Dette var for å vise dialoger.
  • Jeg har ikke implementert login på backend.
  • Jeg brukte openweatherservice sitt API til å få nåværende temperatur i Bud.
